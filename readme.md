# A high available document database based on Raft Algorithm, inspired by [tiedot](https://github.com/HouzuoGuo/tiedot/wiki/Tutorial) and [Raft](https://raft.github.io/)

# how to run 

1. install go
2. go build -o server main.go
3. ./server
